# USSForceGestureRecognizer

USSForceGestureRecognizer is an experimental force gesture recognizer for iOS. It is written in Swift 2.0 and supports iOS 9 and onwards.

Because we currently cannot [3D Touch APIs](https://developer.apple.com/library/prerelease/ios/documentation/UserExperience/Conceptual/Adopting3DTouchOniPhone/3DTouchAPIs.html#//apple_ref/doc/uid/TP40016543-CH4-SW1)
